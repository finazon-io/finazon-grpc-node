# Hello world (javascript)

## How to run

Install dependencies:
```bash
npm install
```

Set correct api key into API_KEY variable

Run app:
```bash
npm run start
```

Example output:
```json
{
  "resultList": [
    {
      "code": "sip_non_pro",
      "name": "Real-Time Stock API: Stock Market Data"
    }
  ]
}
```
