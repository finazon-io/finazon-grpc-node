# Tickers (typescript)

## How to run

Install dependencies:
```bash
npm install
```

Set correct api key into API_KEY variable

Run app:
```bash
npm run start
```

Example output:
```json
{
  "resultList": [
    {
      "ticker": "$ADS/ETH",
      "publisher": "uniswap",
      "publisherName": ""
    },
    {
      "ticker": "$ADS/ETH",
      "publisher": "finazon",
      "publisherName": ""
    }
  ]
}
```
